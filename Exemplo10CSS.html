Exemplo 10 - CSS

Regras CSS

Uma regra CSS é aquela que contém seletores, propriedades e valores que quando aplicado a um elemento html, tem a capacidade de alterar seu estilo. Uma regra css sempre estará dentro de {} e possuirá um seletor. Por exemplo: h1 {propriedade:valor;}

Os seletores são divididos em três tipos básicos: seletores de tipo, de id e de classe. Cada um desses seletores tem uma especificidade que determina sua abrangência.

Exemplo

seletor de tipo

Esse seletor é caracterizado por se assemelhar a uma tag html. Ele sempre será representado pela letra da tag. Por exemplo: h1{propriedade: valor}

Seletor de id

Os seletores de id são aqueles que são aplicados aos elementos html com um id específico. Esse seletor começa pelo símbolo #. Por exemplo: #Div1{propriedade: valor}

Seletor de classe

Os seletores de classe são aqueles em que atribuímos ou aplicamos estilos css a elementos html que possuam o atributo class, esse seletor pode ser aplicado a mais de um elemento hrml. Esse seletor sempre começará por um "." seguido do nome da classe. Por exemplo: .classe1{propriedade:valor}
