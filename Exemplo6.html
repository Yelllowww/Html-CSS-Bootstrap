Elementos de cabeçalho/seção

As tags de seção são utilizadas para estruturar a hierarquia de títulos de uma página. Elas são definidas pela letra h1 ao h6.

Quanto menor o número, maior será sua importância. Exemplo:

Listas Ordenadas(ol) e listas não Ordenadas(ul)

Em html, os elementos de mlista são usados para criar listas de itens organizados em uma página web

Lista Ordenada - ol

Para criar uma lista ordenada no html eu devo utilizar a tag ol, juntamente com li, que representa os elementos da lista. Exemplo:

  1. Primeiro Item
    1. Item1.1
    2. Item1.2
      1. Item1.2.1
      2. Item1.2.2
      3. Item1.2.3
      4. Item1.2.4
    3. Item1.3
    4. Item1.4
  2. Segundo Item
  3. Terceiro Item
  4. Quarto Item
  5. Quinto Item
  6. Sexto Item

Lista não ordenada -ul

para criar uma lista não rdenada no html eu devo utilizar a tag ol, juntamente com li, que representa o elementos da lista. exemplos:

  • primerio item
    • Item 1.1
    • Item 1.2
      • Item 1.2.1
      • Item 1.2.2
    • Item 1.3

  • quarto item
  • quinto item
