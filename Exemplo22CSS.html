O que é o Grid Layout

O Grid Layout é bidimensional, permitindo que o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do flexbox que é unidimensional, ordenando os elementos em linha.

Flexgrid ou Grid Layout

O Flexbox e o Grid Layout podem ser utilizados em conjunto. Você pode aplicar o flexbox dentro de um layout que esteja usando o Grid Layout, aproveitando o melhor das duas técnicas.

A escolha entre flexbox e grid layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas de cada layout, aplicando as suas necessidades.
